## 七、上手步骤：购买之后怎么操作

买完之后，搬瓦工会给你发一封邮件，包含服务器的 IP 地址、SSH 端口（不是标准的22端口）、root 密码这些信息。

然后你有两种管理方式：

一是通过 **KiwiVM 控制面板**，这是搬瓦工自家的后台，可以重装系统、切换机房、查看流量用量、重启服务器等。入口在官网登录后的 "Services" → "My Services" 里找到对应的 VPS，点 "KiwiVM Control Panel" 就能进去。

二是用 **SSH 客户端**（比如 Windows 上的 MobaXterm 或者 Xshell，Mac 直接用终端）连接服务器。注意端口要填 KiwiVM 里显示的实际端口，不要默认写22。

机房切换也是在 KiwiVM 面板里操作，找到 "Migrate to another DC" 的入口，选择目标机房，点确认，几分钟之内就完成了。建议切换前先做个快照备份，虽然实际上切换过程中数据一般不会丢，多个备份总是安心一些。
